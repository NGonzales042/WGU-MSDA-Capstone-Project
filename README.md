# WGU-MSDA-Capstone-Project
Capstone project for MS Data Analytics

## Overview
This project develops a predictive model to identify employees at high risk of burnout using logistic regression.

## Data Source
https://dataverse.harvard.edu/dataset.xhtml?persistentId=doi:10.7910/DVN/VG6KQD 

## Methods
- Data cleaning and preprocessing
- Feature selection via backward stepwise elimination
- Logistic regression modeling
- Model evaluation using confusion matrix and odds ratios

## Tools
- Python
- pandas, scikit-learn, statsmodels
- Jupyter Notebook
